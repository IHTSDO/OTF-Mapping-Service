This demonstration app is loaded with sample mapping projects intended to demonstrate a few common mapping use cases and various aspects of the workflow and other functionality.

Included are a local allergy code set mapped to SNOMEDCT, a local medication code set mapped to RXNORM. Additionally, samples of the SNOMEDCT to ICD10 projects are also loaded to demonstrate more complex workflow.

Start by viewing the demo video. We are happy to provide a guided demo as well as access to this environment so you can play around. Email us at info@westcoastinformatics.com
