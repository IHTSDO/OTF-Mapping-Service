Specialist Dashboard Help

Overview

The Specialist Dashboard displays the map tools that are available to those map editors who have the specialist role. Features include the ability to assign oneself mapping work and to provide a list of assigned work and its current state.

Details

The Specialist Dashboard provides the following tools:



Project Details - displays metadata about the mapping project

Map Records - displays all records within the project that are published or ready for publication




Available Work - allows the map specialists to assign themselves work from the pool of available concepts

Assigned Work - allows the map specialists to view work that is assigned them and to view what state of completion it is in


    Assignment Procedure
  1. Assign individual concepts by clicking on the diagonal download icons
  2. Assign batches of concepts by clicking on the Claim Batch button
  3. Filter available concepts by entering a search term in the text input box

    4. Assigned Work Features
  4. Click on the pencil icon to create a mapping record for the assigned concept
  5. Click on the return icon to return the concept to the pool of available concepts
  6. Click on the trash can icon to return multiple concepts to the pool of available concepts
  7. Filter assigned concepts by entering a search term in the text input box
  8. Filter by workflow status by clicking one of the radio buttons - all, new, editing and finished

    9. Assignment Icons

    Displays a pencil if this record has not yet been edited

    Display a pushpin if this record has previously been edited

    Display a saved icon if this record has been marked finished


Recently Edited - allows the map specialists to view the records they have edited in reverse chronological order
